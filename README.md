Limitation de vitesse en France

Ce dépôt contient une collection d’images représentant les limitations de vitesse en France, sous forme de panneaux stylisés et réalistes.

Ces visuels sont conçus pour être directement réutilisables dans des contextes pédagogiques, analytiques ou de présentation.

🎯 Objectif

Fournir des assets simples, cohérents et prêts à l’emploi pour illustrer les règles générales du Code de la route français.

📦 Contenu

Les images couvrent les limitations de vitesse standards.

Voie de circulation : Vitesse maximale

- Autoroute : 130 km/h

- Route à 2 chaussées séparées par 1 terre-plein central : 110 km/h

- Section de route comportant au moins 2 voies affectées à un même sens de circulation : 90 km/h

- Route à double sens, sans séparateur : 80 km/h

- Agglomération : 50 km/h

- Zone 30 : 30 km/h (à été ajoutée pour les zones piétonnes principalement)

🖼️ Variantes disponibles

Chaque panneau est disponible en 2 versions : Avec texte explicatif → pour usage pédagogique Exemple : 30t.png, 50t.png Sans texte (panneau seul) → pour intégration graphique ou UI Exemple : 30.png, 50.png

👉 Convention :

VITESSE.png → panneau seul

VITESSEt.png → panneau + texte (t = texte)
